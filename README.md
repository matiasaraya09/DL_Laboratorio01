# Laboratorio 01: Predicción de Deterioro Cognitivo
Este laboratorio implementa modelos de clasificación multiclase (Bagging, Boosting y Stacking) para predecir el nivel de deterioro cognitivo.
## Estructura del lab
* `data/`: Contiene los datos (dataset)
* `notebooks/`: Cuaderno con información del lab
* `src/`: Código principal
* `outputs/`: Gráficos y resultados generados

## Cómo ejecutar el código
1. Crea un entorno de conda con Python 3.10: `conda create -n lab_stacking python=3.10`
2. Activar el entorno: `conda activate lab_stacking`
3. Instalar las dependencias: `pip install -r requirements.txt`
4. Ejecuta el main.py: `cd src` y `python main.py`
